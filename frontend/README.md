# Your Thoughts - Frontend PWA

Mobile-first React PWA for "Your Thoughts".

## Setup

1.  **Install**:
    ```bash
    cd frontend
    npm install
    ```

2.  **Dev**:
    ```bash
    npm run dev
    ```
    App runs at `http://localhost:5173` (by default).

3.  **Build**:
    ```bash
    npm run build
    ```

## Config
-   `VITE_API_BASE_URL`: URL of the backend API. Defaults to `http://localhost:3001`.
